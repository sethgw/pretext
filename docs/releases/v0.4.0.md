# pretext v0.4.0

This release turns the project from a strong layout engine into a more usable reader package.

## Highlights

- Added `EpubReader`, a first-class swipe-based EPUB reader widget built on `PagedReader`
- Added resolved EPUB href targets so TOC entries and internal links can jump to real `DocumentCursor` positions
- Added tappable text links in rendered pages, with internal-book navigation and external-link callbacks
- Normalized chapter-relative EPUB links and image paths during load
- Added an EPUB reader demo to the example app

## Reader Surface

You can now go from raw EPUB bytes to a swipeable reader with:

1. `loadEpub(bytes)`
2. `EpubReader(book: result, ...)`

That surface includes:

- page swiping
- TOC drawer navigation
- reader themes
- progress persistence hooks
- internal link jumps
